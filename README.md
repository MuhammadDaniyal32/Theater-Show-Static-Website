# Theatre Show
#### _Theatre show.com is a website used to provide information about various theatre shows going on and where. The Theatre show.com represents the information all the theatre events of events various languages.It also provide information about various Theatres locations,seating and list of events in that theatre._

Checkout [Theatre show.com](https://codesandbox.io/s/theatreshow-com-j5gtho)
